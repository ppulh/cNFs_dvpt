### Unraveling the development of cutaneous neurofibromas in neurofibromatosis type 1

This repository contains the code to ensure the traceability of the analysis of single-cell RNA Sequencing data from the raw count matrices to the figures in the associated publication. Please open the [index.html](https://ppulh.github.io/cNFs_dvpt/) file in any browser, and navigate through all files in this repository.

The results behind this analysis are published in Acta Neuropathologica Communications:

> Pulh *et al.*, Unraveling the development of cutaneous neurofibromas in neurofibromatosis type 1, 2025, *Acta Neuropathologica Communications*; doi: [10.1186/s40478-025-02075-z](https://doi.org/10.1186/s40478-025-02075-z)
